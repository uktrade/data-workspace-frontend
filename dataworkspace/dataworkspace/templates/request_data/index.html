Request data

Use this service to:

  • tell us about data you'd like to publish
  • request your data be published

Before you start

Enter as much detail as you can. You do not need to complete all the information if you do not know it.

We will ask for:

  • a description of the data
  • what the data will be used for
  • the security classification of the data
  • where it is currently held
  • the information asset owner or information asset manager.
  • licensing details

How long it takes

Completing this form takes about 5 minutes. We will respond to you within 5 working days.
